Graph-based Blockchain Exploration

Fall 2022 - Spring 2023

The Elliptic dataset is a large collection of blockchain activity represented as a graph. This capstone will include three integrated components for storing, analyzing, and exploring this dataset. The first component is a back end consisting of a graph database for storing the dataset. The second component is a front-end consisting of a graph visualization, interactions to explore the graph and applied algorithms (see the third component), and an interface for managing the dataset. The third component is the application of one or more graph-based machine learning algorithms. Some graph learning algorithms may be available in the graph database, but some may be in other libraries (Graph Nets, Deep Graph Library, PyTorch, etc.) or be required to be implemented by the student.
